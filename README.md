# Practica-6
En esta practica vamos a servir una pagina web usando el servidor nginx y docker compose.

Pasos para realizar la practica:

1) Crear un directorio src y poner ahi un archivo index.html con hola mundo

2) Crear el archivo docker-compose.yml y especificar el volumen el directorio src y mapearlo al directorio `/usr/share/nginx/html` 

3) Ejecutar el contenedor con el comando `docker compose up -d` y verificar que todo funciona bien

4) Crear repositorio git, subirlo a github y postear url publico en el foro

Puede auxiliarse de chatgpt para la practica.

Valor 5pts
